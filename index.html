King Oyster (2024-25)

Private Discord Bot for a College Organization

King Oyster is a Discord Bot created specifically for the MCC Cabbages & Kings online Discord server community.
King Oyster is programmed in Node JS, with many capabilities including accessing Google Drive, reading spreadsheets, generating QR codes, detecting spam, reminder system, haiku detection, etc. Documentation for older version of this bot are shared on GitHub.
